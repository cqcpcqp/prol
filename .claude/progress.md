## 本次新增：会话历史管理

### 功能概述

实现了完整的**会话历史持久化系统**，让用户可以：
- 保存每次AI对话的历史记录
- 查看历史会话列表
- 恢复之前的对话
- 导出会话为Markdown

### 技术实现

#### 后端 (Rust)

**文件**: `src-tauri/src/session/mod.rs` (~250行)

**数据结构**:
```rust
Session {
  metadata: {
    id, title, project_path, paradigm,
    created_at, updated_at, message_count,
    code_changes: []
  },
  messages: [{ id, role, content, timestamp }]
}
```

**存储位置**:
```
~/.ai-ide/sessions/
├── {uuid-1}.json
├── {uuid-2}.json
└── ...
```

**核心功能**:
- `SessionManager::save_session()` - 保存会话
- `SessionManager::load_session()` - 加载会话
- `SessionManager::list_sessions()` - 列出所有会话（按时间倒序）
- `SessionManager::export_session_to_markdown()` - 导出为Markdown

**命令**:
- `create_session` - 创建新会话
- `get_sessions` - 获取所有会话
- `get_project_sessions` - 获取项目的会话
- `load_session` - 加载特定会话
- `save_session_message` - 保存消息
- `delete_session` - 删除会话
- `export_session` - 导出会话

### 使用流程

```
用户打开项目
    ↓
自动创建新会话（保存到 ~/.ai-ide/sessions/）
    ↓
每次对话自动保存消息
    ↓
用户可查看历史会话列表
    ↓
点击恢复之前的对话
    ↓
可导出为Markdown分享
```
