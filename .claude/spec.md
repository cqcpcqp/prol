# AI原生IDE 产品规格说明书

## 产品概述
面向产品经理的零配置AI开发环境，让非技术人员也能通过自然语言完成软件开发。

## 核心特性
1. **自带运行时**：IDE管理Python/Node运行时，与本机完全隔离
2. **项目级沙箱**：每个项目有独立的虚拟环境
3. **零本机污染**：不修改系统任何配置
4. **Warp风格UI**：左侧文件树、中间代码展示、右侧AI对话
5. **对话式开发**：产品经理只需要描述需求

## 技术栈
- 桌面框架：Tauri v2
- 前端：React + TypeScript + Tailwind CSS
- 代码高亮：Shiki
- 后端：Rust

## 架构要求
- 运行时安装在 `~/.ai-ide/runtimes/`
- 项目虚拟环境在 `.ai-ide/venv/`
- 完全不修改系统PATH
